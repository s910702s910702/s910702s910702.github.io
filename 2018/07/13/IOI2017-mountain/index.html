IOI2017 Practice Task - MOUNTAINS

Mountains

翻譯自：IOI2017 practice tasks - Mountains

Tahmuras，古代波斯的第三個王，他征服了龐大數量的惡魔軍團。他現在想要把這些惡魔關在 Alborz 山，然後讓剩下的惡魔離開。 Alborz 山的稜線由 n 個頂點組成，第 i 個 (0 ≤ i ≤ n - 1) 頂點的座標為 (i, y[i]) ，其中 i 為長度，y[i] 為高度。

惡魔要被關在不同的地方，且兩個惡魔不可以看到對方，如果他們的視線可以對上的話他們就可以策劃逃跑計畫。兩隻惡魔要被山擋住的條件就是他們之間只要有一個比他們兩點連線還要高的頂點。如下圖，一個惡魔在頂點０那麼他可以看得到在頂點２頂點３的惡魔，但是看不到在頂點３,４,５的惡魔，因為頂點２比頂點０到頂點３,４,５的連線還高。

現在請你幫忙 Tahmuras 找出可以放惡魔的最多數量。

實作細節

你必須實作以下的程式，你的程式將會被每個 case 呼叫一次。

  • y : 一個長度為 n 的整數陣列，儲存每個點的高度。
  • 這個程式會回傳惡魔的最大放置量。

限制

  • 1 ≤ n ≤ 2000,
  • 0 ≤ y[i] ≤ 10 (for all 0 ≤ i < n).

子任務

  1. (20 points) n ≤ 19
  2. (20 points) n ≤ 40
  3. (30 points) n ≤ 300
  4. (30 points) 沒有任何限制

範例評分

評分程式以下格式出讀取資料：
第一行：n
第二行：y[0] y[1] y[2] … y[n − 1]
評分程式會印出 maximum_deevs 回傳的值。
