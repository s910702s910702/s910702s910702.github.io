IOI2017 Practice Task - COINS

硬幣

翻譯自：IOI2017 practice tasks - COINS

Jamshid 的敵人 Zahhak，他抓住了 Jamshid 的女兒 Arnavaz 和 Shahrnaz。但是 Zahhak 決定要給姐妹一個機會，只要她們可以解出一個謎題就可以放他們走。

Zahhak 有一個 8 x 8 的西洋棋盤，上面有數字 0 到 63，如下圖所示。64個格子裡都放著一個硬幣，其中一個受到詛咒的硬幣放在標籤為 c 的格子裡，但是它的外觀和和普通的硬幣沒什麼兩樣。每個硬幣都有正面和反面。

Zahhak 邀請兩位姐妹吃飯並敘述了這個謎題：在吃完飯後，姐妹兩人必須要走到不同的房間，Zahhak 先到
Arnavaz 的房間裡並展示盤面還有告訴她 c (受詛咒的硬幣所待的那個格子)是多少，Arnavaz 不能改變硬幣的位置但是可以翻動1 ~ k個硬幣，她也可以翻動同一個硬幣好幾次。完成之後 Zahhak 會再把整個盤面帶到 Shahrnaz的房間。如果 Shahrnaz 可以找出受到詛咒的硬幣，她們就可以被釋放。
姐妹們可以在吃飯時先討論策略，但是之後就沒辦法再討論了。

請你幫助姐妹們解出 Zahhak 的謎題。

實作細節

你必須要實作兩個不同的程式：

  • 這個程式是模擬 Arnavaz
  • b是一個長度為64的整數array，表示 Zahhak 給 Arnavaz 看的棋盤盤面。b[i] (0 <= i <= 63)的值只會是0或1，0和1分別表示硬幣是正面或是反面。
  • c:受到詛咒的硬幣的位置
  • 這個程式必須要回傳一個array，這個 array 代表 Arnavaz 翻動了哪幾個硬幣。這個 array 的程度必須介於1 ~ k。同樣一個硬幣可以被翻動一次以上。
  • 這個程式是模擬 Shahrnaz
  • b是一個長度為64的整數array，表示在 Arnavaz 翻動後的棋盤盤面。
  • 程式必須要回傳 c，表示受詛咒硬幣的位置。

總共會有Ｔ個情況。每個情況評分者會先去呼叫 coin_flips，然後再根據回傳的值去更動盤面，之後再去呼叫 find_coin。請注意判定系統是使用不同的判定程式分別來呼叫你的兩個程式，第一個判定程式會一個情況呼叫 coin_flip 一次，呼叫 find_coin 是放在第二個判定程式中。你的程式必須和

限制

  • 1 <= T <= 1000
  • 0 <= c <= 63

子任務

  1. (10 points) c < 2, k = 1
  2. (15 points) c < 3, k = 1
  3. (10 points) k = 64
  4. (15 points) k = 8
  5. (50 points) k = 1
